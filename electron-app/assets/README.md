# Assets

Place these icon files here before building:

- `icon.ico`         — Windows tray icon (256×256, ICO format)
- `icon.icns`        — macOS tray icon (ICNS format, multiple sizes)
- `icon.png`         — Fallback PNG (256×256)
- `icon-recording.png` — Red variant shown during recording (256×256)

You can convert `logo.png` from the frontend public folder:
- Use https://icoconvert.com/ for .ico
- Use https://cloudconvert.com/ for .icns
- Resize to 256×256 PNG for icon.png and icon-recording.png
  (add a small red dot overlay to icon-recording.png)
